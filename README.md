# コインベットゲーム ｜ 資金推移シミュレーター

コインの表裏（勝率50%・配当1倍）で、**マーチンゲール / 固定比率ベット / 任意ベット**などの資金推移を体感的に検証できる単一HTMLのブラウザアプリです。インストール不要・スマホ対応。

## 🎮 使い方（公開URL）

GitHub Pages で公開されています。スマホ・PCのブラウザでそのまま開けます。

> `https://<ユーザー名>.github.io/coin-bet-game/`

ローカルで使う場合は `index.html` をブラウザで開くだけです（サーバー不要）。

## 主な機能

- **初期設定**: 初期所持金・初回賭け金（0円スタート可）
- **賭け金の組み立て**: `×2` / `＋1%・＋5%（初期/現在基準）` / 任意入力
- **戦略の検証**
  - マーチンゲール: 負けたら `×2`、勝ったら `初回額に戻す`
  - 固定比率: 毎回 `＋％(現在)` で所持金の一定割合を賭ける
  - 任意ベット: 自由入力
- **利確プール**: `💰 余剰金をプール` で初期額を超えた分を確保し所持金を初期額に戻す／`↩ プールから戻す` で復帰
- **可視化**: 3Dコインフリップ演出、総資産の推移グラフ、勝率・最大ドローダウン・最大連敗などの統計
- **履歴コピー**: ワンタップでクリップボードへ（Excel等に貼り付け可）
- **レスポンシブ**: スマートフォン対応

## 技術

- 単一HTML（HTML + CSS + Vanilla JS、依存ライブラリなし）
- グラフは Canvas 2D で自前描画
